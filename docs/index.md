BMad Method (**B**uild **M**ore **A**rchitect **D**reams) is an
agent-assisted software-development framework. It provides specialized AI
agents, guided workflows, and planning tooling that adapt to project
complexity.

It works with any AI coding assistant that supports custom system prompts
or project context (Claude Code, Cursor, Codex CLI, etc.).

installed, invoke the `bmad-help` skill from your AI tool for
context-aware guidance.
